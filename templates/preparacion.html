Selección y Preparación de Datos

Fuentes de Datos

La base de datos proviene de Kaggle (Cost_of_Living_Index_by_Country_2024). Incluye más de 100 países con valores estandarizados de costo de vida, renta, canasta básica, restauración y poder adquisitivo.

Limpieza y Calidad de Datos

  • Eliminación de valores faltantes y revisión de inconsistencias: Se eliminaron registros con datos incompletos y se corrigieron discrepancias en los valores.
  • Estandarización de nombres de países: Se homogenizaron los nombres de los países para evitar duplicidades y asegurar la consistencia en el análisis.
  • Aseguramiento de datos limpios y comparables: Se verificó la integridad de los datos, asegurando que todos los valores fueran precisos y comparables entre sí.

Transformación de Datos

  • Creación de una variable categórica segmentando países según su costo de vida:
    • Alto: >70
    • Medio: 40-70
    • Bajo: <40
  • Cálculo de promedios y referencias a la media global: Se calcularon promedios para cada índice y se compararon con la media global para contextualizar los hallazgos.
